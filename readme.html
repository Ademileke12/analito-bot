üß† How the AI layer works

Analito applies lightweight heuristics on token lists returned by the AURA API to generate insights. The built-in intelligence currently:

  • Detects stablecoins (USDT, USDC, DAI) and ETH exposure.
  • Generates a short summary for each network's token list.
  • Aggregates USD balances across wallets and classifies portfolio composition (ETH / stable / altcoins).
  • Provides human-friendly recommendations like "high altcoin exposure -> higher risk".

Future improvements can replace or augment these heuristics with LLM prompts that explain strategy choices, risk tradeoffs, and suggested rebalances.
